RSA加密算法简要介绍与实现

2021-08-25 Comments

先描述以下RSA算法。

RSA 公钥加密体制由 R.Rivest、A.Shamir 和 L.Adleman 于 1977 年提出，是最著名的公
钥密码，能够抵抗到目前为止已知的绝大多数密码攻击，已被 ISO 推荐为公钥数据加密标
准，RSA 密码的安全性基于数学上的大数分解问题：将两个大素数 p、q 相乘十分容易，但
对其乘积 N = pq 作因子分解却极其困难。在本实验中，假设 p、q 均为 len 比特素数，具体
描述如下。

  1. 系统参数
    选取大素数 p、q，均为 len 比特素数(即 2len-1≤p，q < 2len 且 p、q 为素数)
    计算乘积 N = pq，约为 2*len 比特
    计算 N 的欧拉函数值φ(N) = (p-1)(q-1)
  2. 密钥生成
    公钥(N，e)：其中 e 满足 1 < e < φ(N)且 gcd(e，φ(N)) = 1
    私钥(N，d)：其中 d 满足 1 < d < φ(N)且 ed ≡1(modφ(N))，即 d = e-1modφ(N)
  3. 加解密
    对于明文 m∈[1，N)，加密后得到密文 c = me(mod N)
    对于密文 c∈[1，N)，解密后得到密文 m’= cd (mod N)
